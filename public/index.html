Help & Usage

Getting Started

Type any coding task in the input box and press Enter. The agent reads, writes, searches, and runs code — it does not just chat.

Example: create a hello.py that prints "hello world" and run it

Switching Models

Use the dropdown in the header. Two types:

  • Presets — pre-configured models from presets.json
  • Providers — select a provider, type any model name, click Apply

Supported providers: OpenRouter, Google, Groq, DeepSeek, Mistral, Ollama, LM Studio, Llama.cpp

Header Buttons

🛡️Safe mode — restricts shell commands to a whitelist. ON = green, OFF = yellow
📋Sessions — opens panel to create, switch, or manage conversations. Sessions are independent; each has its own history and model
📄 CopyCopy session — copies entire conversation as formatted text to clipboard
ResetClears conversation history. Agent loses context of previous messages
?Opens this help panel

Toolbar

Allow pathGrant the agent access to files outside the default workspace. Type a path and click Allow

File Changes (Diff Viewer)

When the agent modifies an existing file via write_file, a diff view appears showing exactly what changed:

  • Green lines (+) = added content
  • Red lines (-) = removed content
  • Gray lines = unchanged context (up to 3 surrounding lines)

Diffs are computed line-by-line and capped at 300 lines. The file path and change summary (+N / -M) appear in the header bar.

Important Notes

  • The agent works autonomously — it reads files, runs commands, edits code without your manual help
  • All file operations are restricted to the allowed directory (default: ./workspace)
  • Free-tier API rate limits cause automatic retries with exponential backoff
  • Each session is independent — multiple browser tabs have separate conversations
  • Model switching is instant — no restart needed
  • Tool output is truncated to keep context clean (configurable via MAX_TOOL_RESULT_LENGTH)

Slash Commands (Chat Input)

Type these in the chat input to control the agent:

/model <n>Switch to preset number n
/safeToggle safe mode
/allow <path>Grant path access
/resetClear conversation history
/activeShow current active model
/modelsList all available presets
/mcp listList MCP servers
/mcp toggleDisable/enable MCP tools
/lspToggle LSP code understanding
/exitEnd the session
