در اینجا یک راهنمای کامل، بدون نقص و قدم‌به‌قدم برای ساخت و اجرای این ایجنت در **پاورشل ویندوز (PowerShell)** ارائه شده است. تمام دستورات و کدها دقیقاً برای این محیط بهینه‌سازی شده‌اند.

### پیش‌نیازها
قبل از شروع، مطمئن شوید که **Node.js** (نسخه ۱۸ یا بالاتر) روی سیستم شما نصب است. برای بررسی، در پاورشل تایپ کنید:
```powershell
node --version
```
همچنین به یک **کلید API** از سایت [openrouter.ai](https://openrouter.ai) نیاز دارید (ایجاد حساب رایگان است).

---

### مرحله ۱: ایجاد ساختار پوشه‌ها
در پاورشل، دستورات زیر را یکی‌یکی اجرا کنید تا پوشه‌های پروژه و محل کار ایجنت ساخته شوند:

```powershell
mkdir coding-agent-free
cd coding-agent-free
mkdir src
mkdir src\tools
mkdir workspace
```

### مرحله ۲: راه‌اندازی پروژه و نصب پیش‌نیازها
در پوشه `coding-agent-free`، دستورات زیر را برای ایجاد فایل پیکربندی و نصب کتابخانه‌ها اجرا کنید:

```powershell
npm init -y
npm install openai zod dotenv
npm install -D ts-node typescript @types/node
```
*(نصب بخش دوم که با `-D` شروع شده، برای اجرای کدهای TypeScript در محیط توسعه است).*

### مرحله ۳: ایجاد فایل‌های پیکربندی
در پوشه اصلی پروژه (`coding-agent-free`)، دو فایل زیر را ایجاد کنید. (می‌توانید از ویرایشگر کد مثل VS Code یا حتی Notepad ویندوز استفاده کنید).

**فایل اول: `tsconfig.json`**
این فایل را ایجاد کرده و محتوای زیر را در آن قرار دهید:
```json
{
  "compilerOptions": {
    "target": "ES2022",
    "module": "commonjs",
    "moduleResolution": "node",
    "esModuleInterop": true,
    "strict": true,
    "outDir": "./dist",
    "rootDir": "./src",
    "forceConsistentCasingInFileNames": true,
    "skipLibCheck": true
  },
  "ts-node": {
    "esm": false
  }
}
```

**فایل دوم: `.env`**
این فایل را ایجاد کرده و کلید API خود را در آن قرار دهید:
```env
OPENROUTER_API_KEY=YOUR_API_KEY_HERE
OPENROUTER_MODEL=nvidia/nemotron-3-super-120b-a12b:free
ALLOWED_DIR=./workspace
```
*(به جای `YOUR_API_KEY_HERE` کلید واقعی خود را از سایت OpenRouter قرار دهید).*

### مرحله ۴: ایجاد ابزار مدیریت فایل
در پوشه `src\tools`، فایلی به نام `fileManager.ts` ایجاد کنید و کد زیر را در آن قرار دهید:

```typescript
import { z } from 'zod';
import * as fs from 'fs/promises';
import * as path from 'path';
import * as dotenv from 'dotenv';

dotenv.config();

const allowedDir = path.resolve(process.env.ALLOWED_DIR || './workspace');

function sanitizePath(filePath: string): string {
  const resolvedPath = path.resolve(filePath);
  if (!resolvedPath.startsWith(allowedDir)) {
    throw new Error('Access denied: Path is outside the allowed workspace directory.');
  }
  return resolvedPath;
}

export const tools = [
  {
    type: 'function' as const,
    function: {
      name: 'read_file',
      description: 'Reads the contents of a file from the workspace.',
      parameters: {
        type: 'object',
        properties: {
          path: {
            type: 'string',
            description: 'Relative path to the file within the workspace.',
          },
        },
        required: ['path'],
      },
    },
  },
  {
    type: 'function' as const,
    function: {
      name: 'write_file',
      description: 'Writes content to a file. Creates parent directories if needed.',
      parameters: {
        type: 'object',
        properties: {
          path: {
            type: 'string',
            description: 'Relative path to the file within the workspace.',
          },
          content: {
            type: 'string',
            description: 'The text content to write to the file.',
          },
        },
        required: ['path', 'content'],
      },
    },
  },
  {
    type: 'function' as const,
    function: {
      name: 'list_files',
      description: 'Lists files and folders in a directory within the workspace.',
      parameters: {
        type: 'object',
        properties: {
          directory: {
            type: 'string',
            description: 'Relative path to the directory. Defaults to root workspace.',
          },
        },
        required: [],
      },
    },
  },
  {
    type: 'function' as const,
    function: {
      name: 'delete_file',
      description: 'Deletes a file from the workspace.',
      parameters: {
        type: 'object',
        properties: {
          path: {
            type: 'string',
            description: 'Relative path to the file to delete.',
          },
        },
        required: ['path'],
      },
    },
  },
];

export async function executeTool(name: string, args: any): Promise<any> {
  switch (name) {
    case 'read_file': {
      const fullPath = sanitizePath(args.path);
      const content = await fs.readFile(fullPath, 'utf-8');
      return content;
    }
    case 'write_file': {
      const fullPath = sanitizePath(args.path);
      await fs.mkdir(path.dirname(fullPath), { recursive: true });
      await fs.writeFile(fullPath, args.content, 'utf-8');
      return `File written successfully: ${args.path}`;
    }
    case 'list_files': {
      const dirPath = sanitizePath(args.directory || '.');
      const entries = await fs.readdir(dirPath, { withFileTypes: true });
      return entries
        .map((e) => `${e.isDirectory() ? '[DIR]' : '[FILE]'} ${e.name}`)
        .join('\n');
    }
    case 'delete_file': {
      const fullPath = sanitizePath(args.path);
      await fs.unlink(fullPath);
      return `File deleted successfully: ${args.path}`;
    }
    default:
      throw new Error(`Unknown tool: ${name}`);
  }
}
```

### مرحله ۵: ایجاد حلقه چت تعاملی ایجنت
در پوشه `src`، فایلی به نام `agent.ts` ایجاد کنید و کد زیر را در آن قرار دهید:

```typescript
import * as readline from 'readline/promises';
import { stdin, stdout } from 'process';
import OpenAI from 'openai';
import * as dotenv from 'dotenv';
import { tools, executeTool } from './tools/fileManager';

dotenv.config();

const apiKey = process.env.OPENROUTER_API_KEY;
if (!apiKey) {
  console.error('ERROR: OPENROUTER_API_KEY is not set in .env file');
  process.exit(1);
}

const client = new OpenAI({
  baseURL: 'https://openrouter.ai/api/v1',
  apiKey: apiKey,
  defaultHeaders: {
    'HTTP-Referer': 'https://github.com',
    'X-Title': 'coding-agent-free',
  },
});

const model = process.env.OPENROUTER_MODEL || 'meta-llama/llama-3.1-8b-instruct:free';

const systemPrompt = `You are a helpful coding assistant. You can read, write, list, and delete files in the user's workspace.
When asked to perform file operations, use the available tools.
Always explain briefly what you are doing before and after each operation.
Keep code clean and well-commented.
If a task requires multiple steps, do them one by one and explain each step.`;

interface ChatMessage {
  role: 'system' | 'user' | 'assistant' | 'tool';
  content: string;
  tool_call_id?: string;
  name?: string;
}

async function runAgentLoop(messages: ChatMessage[]): Promise<ChatMessage> {
  const response = await client.chat.completions.create({
    model: model,
    messages: messages as any,
    tools: tools as any,
    tool_choice: 'auto',
  });

  const choice = response.choices[0];
  const assistantMessage = choice.message;

  if (assistantMessage.tool_calls && assistantMessage.tool_calls.length > 0) {
    const toolCalls = assistantMessage.tool_calls;
    for (const toolCall of toolCalls) {
      const functionName = toolCall.function.name;
      const functionArgs = JSON.parse(toolCall.function.arguments);

      console.log(`  🔧 Calling tool: ${functionName}(${JSON.stringify(functionArgs)})`);

      try {
        const result = await executeTool(functionName, functionArgs);

        messages.push({
          role: 'tool',
          tool_call_id: toolCall.id,
          content: String(result),
        });
      } catch (err: any) {
        messages.push({
          role: 'tool',
          tool_call_id: toolCall.id,
          content: `Error: ${err.message}`,
        });
      }
    }

    return runAgentLoop(messages);
  }

  return assistantMessage as ChatMessage;
}

async function startChat() {
  console.log('=================================================');
  console.log('🤖 Interactive Coding Agent (OpenRouter)');
  console.log(`📂 Workspace: ./${process.env.ALLOWED_DIR || 'workspace'}`);
  console.log(`🤖 Model: ${model}`);
  console.log('=================================================');
  console.log('Enter your request. Type "exit" to quit.\n');

  const rl = readline.createInterface({ input: stdin, output: stdout, prompt: 'You: ' });

  const messages: ChatMessage[] = [
    { role: 'system', content: systemPrompt },
  ];

  rl.prompt();

  for await (const line of rl) {
    const input = line.trim();
    if (input.toLowerCase() === 'exit') {
      console.log('Exiting...');
      rl.close();
      break;
    }
    if (!input) {
      rl.prompt();
      continue;
    }

    messages.push({ role: 'user', content: input });

    try {
      console.log('\n⏳ Thinking...\n');

      const result = await runAgentLoop(messages);
      messages.push(result);

      console.log(`\nAgent: ${result.content}\n`);

      if (result.usage) {
        console.log(`[Usage: ${JSON.stringify(result.usage)}]\n`);
      }
    } catch (err: any) {
      console.error(`Error: ${err.message || err}`);
    }

    rl.prompt();
  }
}

startChat();
```

### مرحله ۶: اجرای ایجنت
اکنون همه چیز آماده است. در پاورشل، دستور زیر را اجرا کنید:

```powershell
node node_modules/ts-node/dist/bin.js src/agent.ts
```

**خروجی مورد انتظار:**
```text
=================================================
  Interactive Coding Agent (OpenRouter)
  📂 Workspace: ./workspace
  🤖 Model: nvidia/nemotron-3-super-120b-a12b:free
=================================================
Enter your request. Type "exit" to quit.

You:
```

حالا می‌توانید به زبان فارسی یا انگلیسی با آن صحبت کنید. **مثال‌هایی برای تست کردن:**

1.  `یک فایل پایتون به اسم hello.py بساز که hello world چاپ کند.`
2.  `لیست فایل‌های workspace را نشان بده.`
3.  `محتوای فایل hello.py را بخوان و به جای hello world بنویس سلام دنیا.`
4.  `exit`

در طول مکالمه، خواهید دید که ایجنت به‌طور خودکار از مدل رایگان انتخاب شده استفاده کرده و از ابزارهای خواندن، نوشتن، لیست و حذف فایل استفاده می‌کند، در حالی که سابقه گفتگوهای قبلی را کاملاً به خاطر می‌سپارد.
